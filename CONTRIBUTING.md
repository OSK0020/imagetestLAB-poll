# Contributing to AI Models Laboratory

Thank you for your interest in contributing! 

This repository is a **Public Source Preview & Architecture Reference**. To protect the integrity and proprietary nature of the production UI design and 3D assets, the active codebase is hosted privately.

## How to Contribute

1. **Feedback & Bug Reports**: If you find an issue with the API routes or the core audio engines, please open an issue describing the bug.
2. **Feature Suggestion**: Submit an issue detailing your feature idea. We will review it and consider implementing it in the private master codebase!
3. **Pull Requests**: Pull requests are currently closed for the master UI, but you are welcome to submit PRs for fixing typos or improving comments in the API documentation files.

Thank you for respecting the intellectual property of this project!
